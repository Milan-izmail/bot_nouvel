# Bad Product Patterns

## Pattern 1

Погано:
"Композиція виконана в ніжній рожевій гамі та має пишну форму."

Чому погано:
Це опис фото, а не продаж емоції.

Краще:
"Композиція створена для привітання, яке має виглядати продумано: ніжна рожева гама, щільна салонна форма і подарункова подача в коробці створюють відчуття турботи без зайвої демонстративності."

## Pattern 2

Погано:
"Букет виглядає свіжо і дуже виразно."

Чому погано:
Це порожня фраза.

Краще:
"Букет добре працює як світлий жест уваги, не надто урочистий, але достатньо помітний, щоб залишити після себе тепле враження."

## Pattern 3

Погано:
"Чудово підійде для будь-якого свята."

Чому погано:
Занадто загально.

Краще:
"Підійде для дня народження, подяки, побачення або привітання, де важливо передати увагу без зайвих слів."

## Weak Example To Reject

"Велика квіткова композиція в круглій коробці виконана у ніжній рожевій гамі та має пишну, щільну форму. Світло-рожева коробка з фірмовим оформленням Nouvel Amour робить подачу зібраною й виразною, а сама композиція звучить м'яко, романтично та дуже камерно."

Reject this because it is generic, describes the photo, does not sell emotion, has no use scenario, and uses the artificial phrase "композиція звучить".
